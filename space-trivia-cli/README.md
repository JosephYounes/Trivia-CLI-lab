# Space Trivia CLI

Welcome to the Space Trivia CLI application! This project is a command-line interface trivia game that tests your knowledge about space and astronomy.

## Features

- A set of trivia questions related to space.
- User-friendly command-line interface.
- Score tracking to see how well you did.

## Installation

To get started, clone the repository and navigate into the project directory:

```bash
git clone <repository-url>
cd space-trivia-cli
```

Then, install the necessary dependencies:

```bash
npm install
```

## Usage

To run the Space Trivia CLI application, use the following command:

```bash
node src/index.js
```

You will have 60 seconds to answer as many questions as possible. Good luck!
